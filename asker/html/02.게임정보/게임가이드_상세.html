가이드 템플릿

이 문서는 퍼블리싱 게임 사이트의 게임 가이드 템플릿의 기본 레이아웃 설명을 담고 있습니다. 실제 가이드 페이지의 이런 식으로 이루어질 것이라는 걸 확인하실 수 있도록 작업하다 보니, 주저리주저리 설명이 길어질 수 있습니다. 바로 지금처럼요!

 

각 레이아웃을 구성하는 HTML 태그는 소스보기나, 브라우저에서 제공하는 개발자 도구를 통해 확인하실 수 있습니다. 가이드 템플릿에 필요한 태그는 모두 guide_ 로 시작하는 클래스를 가지고 있습니다.

 

이 문서는...

이 문서는 트리니티2 스타일과 가로폭을 기준으로 작업되었습니다.

 

 

CSS와 style 정의

이 예제 파일에서는 별도의 CSS 파일을 링크했으나, 실제 게임 사이트에 서비스 될 경우에는 필요한 style이 해당 게임의 CSS 파일에 개별적으로 들어갑니다. 웹퍼블리싱팀의 사이트 작업 담당자는 이 파일을 참고해 필요한 style을 디자인에 맞게 변경하여 해당 게임 사이트의 CSS 파일에 들어갈 수 있도록 작업해야 합니다.

 

 

문단의 구분과 간격 조절

TinyMCE 에디터는 기본적으로 Enter 키를 입력했을 경우 문단을 P태그로 구분합니다. 가이드 작업자는 단락의 간격을 Enter 키를 입력하여 적당히 보기좋은 간격을 유지해야 합니다. 같은 문단내에서 줄바꿈이 필요할 경우 Shift + Enter 키로 BR 태그를 입력하여 줄바꿈을 표시합니다.

 

 

다른 페이지로의 링크

다른 페이지로 링크 스타일은 이렇습니다.

 

 

이미지 삽입

모든 이미지는 div.guide_image 로 감싸서 들어갑니다. 아래 예제는 기본적인 이미지 삽입 레이아웃을 보여줍니다.

 

- 이미지는 CSS에서 크기를 조절하지 않습니다.

- 레이아웃보다 큰 이미지를 삽입 시 에디터에서 가로x세로 간격을 지정해주는 옵션이 필요합니다.

- 이미지에는 기본적으로 해당 게임 사이트 디자인에 맞는 테두리가 무조건 들어갑니다.

- 이미지에는 기본적으로 간격이 들어가지 않습니다. 상하 간격이 필요할 경우 Enter 키을 치세요!

왼쪽의 이미지를 감싸는 텍스트

div.guide_preset_left 는 왼쪽에 이미지를 두고, 텍스트를 우측에 감싸듯이 배열합니다. 이미지의 크기는 CSS에서 제한하지 못합니다. 레이아웃보다 큰 이미지를 삽입할 필요가 있을 경우 에디터에서 가로x세로 크기를 지정해 줄 수 있는 옵셥이 필요합니다.

 

이하 왼쪽/오른쪽 레이아웃은 최상단 DIV의 클래스명만 차이가 날 뿐 구성과 사용법은 같습니다.

 

이미지를 감싸는 레이아웃을 벗어나서 텍스트를 입력하면 이렇게 배치됩니다.

오른쪽의 이미지를 감싸는 텍스트

div.guide_preset_right 는 이미지를 텍스트 우측으로 배열합니다. div.guide_preset_right 이하의 태그 구성은 왼쪽에 이미지를 배치할 경우와 동일합니다.

 

여러 이미지를 수직적으로 배치할 경우

이 레이아웃은 여러 이미지를 우측에 수직으로 배열하고 그 옆에 텍스트를 배치하는 레이아웃입니다. 이 경우에는 수직으로 배치할 여러개의 div.guide_image 전체를 하나의 div.guide_image_vertical 로 감싸줄 필요가 있습니다.

 

또한 이렇게 수직으로 연속된 이미지를 배치하는 방법은 왼쪽의 이미지를 감싸는 텍스트 레이아웃에서도 동일하게 사용할 수 있습니다.

표 스타일

표의 내용이 유동적이고 컬럼의 갯수를 특정할 수 없기 때문에, 표의 가로폭을 특별히 지정해 주지 않습니다.

 

- 컬럼의 스타일이 두 종류이고 배경색이 있을 경우 td.guide_th로, 배경색이 없을 경우는 td.guide_td로 구분해줍니다.

- 컬럼의 스타일을 편집 시 변경가능하므로 TH, TD 태그를 구분해서 사용하지 않고 TD 태그만 사용합니다.

- IE에서는 TABLE 에 TBODY 태그가 없으면 동적으로 TD 를 삽입할 때 버그가 있을 수 있으므로, TBODY 태그가 들어갑니다.

- 컬럼 안에서의 좌/우/중앙 정렬 옵션은 에디터의 기본 기능을 사용하면 될 것 같습니다.
